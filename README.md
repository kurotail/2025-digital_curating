# 單本框架
## 結構

捲動動畫為主。為手機的尺寸設計為主。

- 網頁設定
  - (預留)返回首頁
  - 深色模式
  - (預留)語言
 
- 基本資料
  - 封面圖
  - 文字資料
    - 書名
    - 標籤
  - 其他資訊
    - 作者
    - 系列卷數資訊
    - 出版社 / 發行商
    - ISBN
    - 發售日期

- 內容資訊
  - 簡介
  - 角色介紹
  - 評價 / 推薦
    - 引用

- 外部連結
  - 購買連結

## 細項
一次顯示一項，網頁設定除外。基本資料向下滑動到底內容資訊會滑動至畫面，依此類推。
### 網頁設定
初始狀態只有上述功能且高度較高，向下捲動後會把書名、作者一併顯示，功能繼續顯示但整個widget高度需降低。

### 基本資料
widget高度較高，能使人第一眼注意。依序放置。其他資訊有兩欄，部分可以用兩欄的空間整合為一欄。書名獨立，不須太大，書名可以左右滑動，不應滑動而超出書名寬度。只會介紹整體，不做卷數導航。封面固定長寬比。

### 內容資訊
文字穿插圖片，圖片應帶有說明。引用使用特別設計顯示。內文圖片不超過畫面。

### 外部連結
文字超連結。
